Welcome!

This is a self-hosted FluidApp - a single page Javascript application for interacting with FluidDB that is hosted on FluidDB itself.

THIS IS A WORK IN PROGRESS (so it doesn't do much really)

Here's what we've got so far:

  • A bunch of namespace/tags defined under namespaces/ntoll/pocketFluidDB that represent the names of the various files needed for this application to run.
  • An object (with ID: 3d8808e5-e546-4db9-91ce-6fa34c2665e0) that has these tags associated with it. The values of these tags are the various files themselves (stylesheets and javascript mostly)
  • I'm using the rather awesome Sammy javascript application framework to help with the "single page" nature of this app.
  • None of this would be possible without the use of Emanuel Carnevale's work on a FluidDB Javascript library.
  • For more information contact me: Nicholas Tollervey

To get started enter your FluidDB username and password in the box above. You should see some raw JSON (pulled from FluidDB) replace this text. Hooray! This proves this app can talk to FluidDB!
